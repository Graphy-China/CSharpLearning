1. working in topic

2. Apply patches from email.
  Generate Patch:
  	   diff
	   format-patch
  Method-1: use apply to deal with pathes that are generated by "diff".
  git apply --check  xyz.patch	# check  if a patch applies cleanly.
  git apply xyz.patch		# apply the patch (apply all or abort all mode)

  Method-2: use am to deal with patches that are generated by "format-patch"
  git am xyz.patch
